# Project Images

Letakkan screenshot project kamu di folder ini dengan nama file:

1. `koperasi-system.jpg` - Screenshot Koperasi System
2. `gacha-simulator.jpg` - Screenshot Gacha Simulator  
3. `outpatient-report.jpg` - Screenshot Outpatient Reporting
4. `fast-claim.jpg` - Screenshot Fast Claim

## Spesifikasi Gambar:
- **Resolusi**: 1200x800px (ratio 3:2)
- **Format**: JPG atau WebP
- **Ukuran**: < 500KB per file
- **Kualitas**: 80-90%

## Tips Screenshot:
- Ambil screenshot bagian paling menarik dari aplikasi
- Pastikan UI terlihat jelas dan bersih
- Gunakan device/browser dengan resolusi tinggi
- Crop untuk fokus pada fitur utama

Jika belum punya screenshot real, bisa gunakan placeholder dulu:
- https://placehold.co/1200x800/14b8a6/ffffff?text=Project+Name
